# Dalit Dignity & Justice Center (DDJC)

A full-stack MERN (MongoDB, Express.js, React, Node.js) web application dedicated to promoting dignity, justice, legal awareness, and social empowerment for marginalized communities.

## 🚀 Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer

## 📁 Project Structure

```text
DDJC/
│
├── client/                 # React Frontend
├── server/                 # Express Backend
├── README.md
└── package.json
```

## 📦 Installation

### Clone Repository

```bash
git clone <repository-url>
cd DDJC
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd ../server
npm install
npm run dev
```
